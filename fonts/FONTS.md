# Fonts Guide

This project embeds fonts from the local `fonts/` directory for reproducible PDF output.

## Bundled open-source fonts (commercial use allowed)

The repository currently includes:

- `Carlito-Regular.ttf`
- `Carlito-Bold.ttf`
- `Carlito-Italic.ttf`
- `Carlito-BoldItalic.ttf`
- `NotoSansCJKsc-Regular.otf`
- `NotoSansCJKsc-Bold.otf`

License:

- Carlito: SIL Open Font License 1.1 (OFL)
- Noto Sans CJK SC: SIL Open Font License 1.1 (OFL)

## Optional compatibility fonts (not bundled)

If you need closer visual compatibility with the original template, you may also provide:

- `Calibri.ttf`
- `Calibri Bold.ttf`
- `Calibri Italic.ttf`
- `Calibri Bold Italic.ttf`
- `PingFangSC-Regular.ttf`
- `PingFangSC-Semibold.ttf`

These are usually proprietary fonts. Only distribute them if you have proper rights.
